Introducing Pyxeled

Transforming normal photographs into aesthetic pixel art. That's Pyxeled.

Inspired by the paper Pixelated Image Abstraction by Gerstner et al., this application leverages intelligent clustering algorithms to optimally render images at lower resolutions and limited color palettes.

Check out an image gallary created using Pyxeled as well as an explanation of the algorithms used here! Feel free also to browse the code for Pyxeled on its public GitHub repository.

I loved the retro low-dimensional feel of this project which garnered no small amount of nastalgia for high school days spent programming video games. The style also served as a welcome relief to the more intense physical simulation of desert sand dunes I worked on previously.

If you think Pyxeled is cool, you might also enjoy this fun blog post from Google AI on applying deep learning to transfer style between images.
